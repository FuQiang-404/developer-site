# 使用钉钉 JSAPI

## 示例说明

钉钉 JS API 是由钉钉容器提供的, 可以由前端代码调用的 API, 用于做一些导航, 页面跳转相关的操作, 下面介绍一些在宜搭页面的场景下一些常用的使用方法当在钉钉内打开宜搭页面时, 钉钉的 JS API 会挂载到 `window.dd` 的全局变量上。
